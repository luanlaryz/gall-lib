# Spec 151: Demo Web UI Diagnosis

## 1. Objetivo

Avaliar se a demo web/UI oferece uma experiência local simples e útil, coerente com a proposta da `gaal-lib`.

---

## 2. Checklist obrigatório

- [ ] UI sobe localmente
- [ ] backend e UI integram corretamente
- [ ] há fluxo textual
- [ ] há fluxo streaming
- [ ] a documentação é clara
- [ ] a experiência é simples o suficiente para demo

---

## 3. Classificação

- `PASS`
- `PARTIAL`
- `FAIL`
